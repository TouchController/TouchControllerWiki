# 标签页：调试

## 说明

- 此标签页属于“通用”分类
- 此标签页中的所有选项均为开关
- 此标签页中的所有选项默认禁用

## 设置项

### 显示指针位置

- 游戏内描述：在屏幕上绘制所有指针。
- TouchController 会在所有获取到的触摸点下方渲染一个回字形指针，并在其右下方渲染一个 ID，这个 ID 表示了当前触摸点是游戏启动以来的第几个触摸点。
- 使用鼠标模拟的触摸点，其 ID 永远为 0。

### 触摸模拟

- 游戏内描述：将鼠标输入模拟为触摸输入。
- 启用后 TouchController 将不再获取触摸信息。
- 启用后会屏蔽鼠标点击，即使没有启用[“屏蔽鼠标点击”](../通用/常规.md#屏蔽鼠标点击)。
